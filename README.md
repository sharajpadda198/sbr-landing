# FarmConnect Landing Page

This is a Vite + React + TypeScript + TailwindCSS landing page for **FarmConnect**.

## Routes
- `/landing` — Public marketing landing page
- `/login` — Placeholder login page (replace with your real auth)

## Structure
- Landing sections: `src/components/landing/*`
- UI primitives (shadcn/ui-style): `src/components/ui/*`

## Contact form

- Shows a success toast + success state

Replace the placeholder email/phone in the contact section when ready.
